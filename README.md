# nginx-mediawiki

Instructions for setting up MediaWiki on nginx on Ubuntu 16.10

Full list of software:

- Ubuntu 16.10
- nginx
- MediaWiki
- MediaWiki templates
- php7.0-fpm
- certbot (letsencrypt)
- Linode for domain stuff

## MediaWiki extensions

- Scribunto
- ParserFunctions
- Cite

## MediaWiki imports

```
Template:Reflist
Template:Snd
MediaWiki:Common.css
MediaWiki:Common.js
```
